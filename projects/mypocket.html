MyPocket - Personal Finance Manager

  • Tech Stack: JavaFX, MySQL, JFoenix library
  • Github URL: Project Link

A desktop application that enables users to manage their expenses by adding income, recording transactions, and setting budgets to avoid exceeding limits.

Users can view their expenses categorized by type (Food, Transportation, Hobbies, etc.) through a pie chart. Additionally, they have the option to automatically set a budget based on their expenses from the previous month to avoid exceeding it.
